## Ընդհանուր նկարագրություն (Overview)

Նպատակն է ստեղծել ժամանակակից, արագ և բարձր արդյունավետությամբ e-commerce վեբ հարթակ, որը հնարավորություն կտա բիզնեսին թվայնացնել իր վաճառքները և ապահովել հաճախորդների համար հարմար ու վստահելի գնումների փորձ։

Հարթակը նախատեսված է`
- ուտելիքների և ապրանքների պրոֆեսիոնալ ներկայացման
- օնլայն պատվերների արագ և պարզ ձևակերպման
- բրենդի վստահության և ճանաչելիության բարձրացման

## Կայքի հիմնական էջեր

- Գլխավոր - Home
- Խոհանոց - Shop page
- Կոմբոներ
- Մեր մասին էջ - About us
- Կապի էջ - Contact page

## 1. ԳԼԽԱՎՈՐ ԷՋ (Home)

### Ֆունկցիոնալ պահանջներ
- Header - մենյու
- Գլխավոր բաննեռի հատված
- Կատեգորիաների ներկայացման հատված
- Ամենավաճառվող կամ ընտրված ապրանքների կարճ ցանկ
- Ակցիաներ, զեղչեր, հատուկ առաջարկներ հատված
- Footer
  - Կոնտակտային տվյալներ
  - Սոցիալական հղումներ

## 2. ԱՊՐԱՆՔՆԵՐԻ ԷՋ (Shop) - Խոհանոց, կոմբոներ

### Ֆունկցիոնալ պահանջներ
- Ապրանքների ցուցակ՝
  - Նկար
  - Անվանում
  - Կատեգորիա
  - Գին (զեղչված գին)
- Սորտավորում
  - Price (ASC / DESC)
  - Newest
  - Popular
- Ֆիլտրեր կախված՝
  - Գնային միջակայքից
  - Կատեգորիաներից
  - Այլ ֆիլտրեր կախված ապրանքի հատկություններից

### 2.1. ԱՊՐԱՆՔԻ ԷՋ (Single Product Page)

Յուրաքանչյուր ապրանքի էջ պետք է ներառի`
- Ապրանքի նկար
  - Multiple images
  - Zoom ֆունկցիոնալ
- Ապրանքի մասին ինֆորմացիա
  - Անվանում
  - Կարճ նկարագրություն
  - Բաղադրություն
  - Քանակա, չափ կամ տեսակը ընտրելու հնարավորություն
  - Հավելումներ անելու հնարավորություն
  - Բացառություններ անելու հնարավորություն
- Գին
  - Ներկայիս արժեք
  - Զեղչված գին
- “Ավելացնել զամբյուղ” կոճակ
- Stock Status - Առկա, Սպառված
- Նմանատիպ ապրանքներ

### 2.2. CHECKOUT (Պատվերի ձևակերպում)

Պատվերի ձևակերպման ձևի դաշտեր`
- Անուն / Ազգանուն
- Հեռախոսահամար
- Էլ. փոստ (Email)
- Քաղաք / շրջան
- Հասցե (շենք, հարկ, բնակարան, դոմոֆոնի կոդ) - քարտեզ
- Վճարման եղանակ (բանկի ինտեգրացիա)
- Առաքման արժեք
- Մանրը նշելու հնարավորություն
- Պատվերի ընդհանուր գումար
- Լրացուցիչ նշումներ պատվերի վերաբերյալ

## 3. About Us էջ

Այս էջում ներկայացվելու է կազմակերպության մասին տեղեկատվություն։
Էջը ներառելու է՝
- Տեքստային բովանդակություն կազմակերպության մասին
- Նկարներ (աշխատանքային գործընթաց կամ այլ)
- Վստահություն ձևավորող տեղեկատվություն

## 4. ԿԱՊԻ ԷՋ

Պետք է պարունակի կապի համար անհրաժեշտ բոլոր տվյալները՝ հեռախոսահամար, էլ. փոստ, հասցե, քարտեզ, սոցիալական հղումներ և կապի ձև (մինիմալ ֆոռմա), որպեսզի այցելուները կարողանան հեշտությամբ կապ հաստատել ընկերության հետ։

## 5. USER ACCOUNT (Customer Profile)

### Ֆունկցիոնալ
- Registration / Login  
  Օգտատիրոջ գրանցում և մուտք (email/phone)
- Order History  
  Պատվերների ամբողջական պատմության դիտում
- Reorder Functionality  
  Նախկին պատվերի արագ կրկնում մեկ սեղմումով
- Address Management  
  Առաքման հասցեների ավելացում, խմբագրում և ջնջում
- Profile Management  
  Անձնական տվյալների դիտում և խմբագրում
- Coupons  
  Կուպոնների կիրառման պատմություն կամ հասանելի կոդերի ցուցակ

## 6. Ադմին պանել (կառավարման համակարգ)

### 1) Ապրանքների կառավարում
- Ավելացնել / խմբագրել / ջնջել ապրանքներ
- Կատեգորիաների կամ ապրանքի տեսակների ավելացում / խմբագրում

### 2) Պատվերների կառավարում
- Պատվերների ցուցակ
- Պատվերի մանրամասների դիտում
- Պատվերի կարգավիճակի փոփոխում

### 3) Ակցիաներ, կուպոններ, զեղչեր
- Զեղչերի կարգավորում, կառավարում
- Կուպոնների կառավարում

### 4) Վերլուծության վահանակ (Analytics Dashboard)
- Վաճառքի վիճակագրություն
- Պատվերների վիճակագրություն ըստ կարգավիճակի
- Ապրանքների վիճակագրություն
- Պահեստի վիճակագրություն
- Հաճախորդների վիճակագրություն
